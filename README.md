# BasiCLI

Support for CLI oriented applications.

## Building

### Requirements

* [Apache Maven](https://maven.apache.org/) 3.6+ (prefer to use included `mvnw`)
* OpenJDK 11+
* BaSH (for `build` script)

### Build

Rebuild the codebase:

```sh
./build rebuild
```
